# Team Sorting Script
This script extracts a group of students, their grade, and a link to their repository from a GitHub Classroom assignment grading .csv spreadsheet. 
## Steps to Use
1. Download the grading spreadsheet from eLearning.
2. Sort it in your desired order (likely alphabetical by last name).
3. Copy the NetIDs of your students into a file named 'students.txt'. 
4. Download the grades for your desired assignment from GitHub Classroom.
5. Open the 'manual_grading' directory. 
6. Run `python RUNME.py` (Windows) or `python3 RUNME.py` (MacOS). 
7. Type your desired output filename in the output field. 
8. Select the output directory, assignment spreadsheet, and student list. 
9. Generate the new .csv file.
